jQuery Facets

Purpose

Every time I've needed faceted search capabilities, I've had to rewrite the JavaScript to do it. This plugin hopes to do away with that. I tried looking for an existing plugin that did what jQuery Facets does, but surprisingly I couldn't find any that offered the functionality that I wanted. Hopefully others will find this plugin useful too.

Overview

Below is the proper way to include the plugin on your page:

<script type='text/javascript' src='jquery.deserialize.js'></script>
<script type='text/javascript' src='jquery.facets.js'></script>

The jQuery Deserialize plugin is necessary if you plan on using the hashtag features of jQuery Facets.

The simplest way to use jQuery Facets is like so:

$("form#facets").facets();

However, this assumes some defaults.

Defaults

Below are all of the options that can be set with jQuery Facets and all of their default values:

{
'searchCont' : '#searchCont', //A selector for where the filtered search results will go. i.e., a div.
'ajaxURL' : $(location).attr('href'), //URL to make ajax requests to for filtered results. Defaults to current URL.
'URLParams' : new Array(), //extra params for the url, such as the original search query or something. takes array in jQuery's serializeArray() format
//i.e.
//[
// {
// "name":"search",
// "value":"brown jackets"
// }
// ]
'ajaxMethod' : 'GET', // POST or GET
'bindType' : 'change',
'bindTypes' : new Array(),
'excludeBindTypes' : new Array(),
'preAJAX' : function () { return true }, //pre-call ajax function, used for validation etc. returns true or false
'postAJAX' : function (data) { $(plugin.data("settings").searchCont).html(data) }, //post-call ajax function
'preHash' : function () {},
'postHash' : function () {},
'hashOptions' : [], //takes a array of hashes of key value pairs. Must be in name value format
'hash' : true, //whether or not to put facet values in url after hashtag #
}
